Тесты для тестирования API учебного сервиса Яндекс Самокат. Его документация: qa-scooter.praktikum-services.ru/docs/

Основа для написания тестов — фреймворк pytest
Директория allure-report - содержит отчет о результатах выполнения тестов, сгенерированный с посмощью фреймворк Allure

Проект состоит из:

Директории methods - содержит методы:
courier_methods - выполняемые методы для работы с курьером
order_methods - выполняемые методы для работы с заказом

Директория tests - содержит в себе поддиректории:
couriers - включает тесты:
- test_create_courier - тесты на проверку создания курьера с разными входными данными
- test_login_courier - тесты на проверку авторизации курьера с разными входными данными
orders - влючает тесты:
- test_create_order - тесты на создание заказа с разными входными данными 
- test_list_orders - тесты на получение списка заказов с разными условиями выбора
